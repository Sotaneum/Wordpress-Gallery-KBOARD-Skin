# Wordpress-Gallery-KBOARD-Skin
Wordpress Gallery KBOARD Skin

## Info

- [Wordpress](https://ko.wordpress.org/)의 [KBOARD](https://www.cosmosfarm.com/products/kboard) 게시판에서만 동작하는 SKIN이며 기본 썸네일 SKIN을 살짝 변형했습니다.

## How to use

1. GallerySkin을 다운로드합니다.
2. KBOARD 게시판 Skin 폴더에 압축을 해제합니다.
3. 관리자 페이지에서 skin을 변경합니다.

## Finally

- 조만간 업데이트를 통해서 최신버전에서 정상작동하도록 하겠습니다.
